# Comparison Sheet versions append workbook history

Rialto will persist the current Quote Comparison sheet view and append an immutable workbook version whenever the normalized sheet view changes. The latest view remains in `comparison_sheet_views` for fast page hydration, while `comparison_sheet_versions` records version number, parent version, source, actor, summary, timestamp, full view JSON, and optional agent proposal JSON. Agent-approved proposals use source `agent-proposal`; ordinary browser edits default to `estimator-edit`; restoring an older version writes a new `restore` version instead of deleting later history. This keeps the estimator-visible sheet as the source of truth for export while preserving enough workbook history to recover from imports, vendor merges, manual edits, and agent edits.
